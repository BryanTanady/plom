## FAQ list

### How do I change the GUI?

You need two tools `qtcreator` and `pyuic5`.

1. Use qtcreator to edit the `qtCreatorFiles/ui_foo.ui` file.
2. Comand line: `pyuic5 ui_foo.ui > client/uiFiles/ui_foo.py`
